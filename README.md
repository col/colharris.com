## colharris.com

[https://github.com/jekyller/jasper2](https://github.com/jekyller/jasper2)

### Prerequisites

- Node v10 (Node: >= v12 kwill break gulp)
- Ruby

### Setup

```
nvm use lts/dubnium
npm install -g gulp
bundle install
```

### Compile Assets

```
gulp
```

### Build

```
bundle exec jekyll build -d ../colharris.com-pages
```
